💡 Insights ejecutivos y recomendaciones

1. Fricción puntual: cash en web

La combinación con peor tasa de éxito es cash + web con 17.8% de rechazo sobre 163 transacciones (vs. 7.7% de fallo global).

🎯 Recomendación: auditar la integración de cash en el canal web antes de la próxima campaña. Revisar logs de error específicos y compatibilidad de versiones del SDK.

2. Diferencia de performance entre canales: web vs api

El canal web procesa transacciones en 794 ms en promedio, mientras que api lo hace en 247 ms — una diferencia de 3.2×.

🎯 Recomendación: investigar el cuello de botella en web (validaciones cliente, render de pasarela, llamadas síncronas). Bajar la latencia impacta directamente en abandono y conversión.

3. Concentración de volumen: Card = 46% del negocio

Card procesa $3,080,527,449 (46% del total económico), con tasa de éxito 88.7%. Riesgo de concentración: una caída de esta pasarela afecta casi la mitad del negocio.

🎯 Recomendación: implementar fallback automático entre pasarelas. Lanzar incentivos puntuales para diversificar el mix hacia PSE y wallet.
